# PDFedit

![License](https://img.shields.io/badge/license-MIT-blue)
![Language](https://img.shields.io/badge/language-Python-3776AB)
![Type](https://img.shields.io/badge/type-Desktop%20App-brightgreen)

A simple, focused PDF editor built with Python. View, edit, merge, split, and annotate PDF files with a clean dark-themed desktop interface.

## Features

- **View PDFs** — Browse pages with zoom and navigation controls
- **Edit Pages** — Reorder, rotate, and delete pages
- **Merge** — Combine multiple PDFs into one document
- **Split** — Extract specific pages or page ranges
- **Annotations** — Add text, highlights, and notes
- **Dark Theme** — Professional dark-themed interface

## Installation

```bash
python PDFedit.py
```

Dependencies auto-install on first run.

## Requirements

- Python 3.8+
